Training Planner:
Автор: Гончаров Владимир Сергеевич

Описание:
Простое GUI-приложение для планирования и учёта тренировок. Позволяет добавлять тренировки с указанием даты, типа и длительности. Реализована фильтрация по типу и дате. Все данные сохраняются в файл trainings.json для последующего использования.

Требования:
Python 3.x
Библиотека tkinter (входит в стандартную поставку Python)

Запуск приложения:
1. Установите Python 3 (если не установлен).
2. Склонируйте репозиторий или скачайте архив.
3. Запустите файл training_planner.py:
4. В открывшемся окне введите данные о тренировке и нажмите «Добавить тренировку».
5. Используйте фильтры для поиска по типу или дате.
6. Данные сохраняются автоматически при добавлении новой тренировки.
